# SH-CHAT-GPT
Open Ai Chat GPT for python Tools

<h3 align="left">INSTALL :</h3>

````bash
pkg update && upgrade -y
pkg install python -y
pkg install git -y
git clone https://github.com/ShTasrif/SH-CHAT-GPT
cd SH-CHAT-GPT
pip install requests
pip install openai
python shgpt.py
````

## How to use 

#### Run :

````bash
shgpt
````
